# crypto-market-intelligence-dashboard
Business intelligence dashboard providing crypto market insights, top movers, volatility analysis, and executive‑level reporting.
As an aspiring data engineer, I am focusing on developing practical, production‑style projects that integrate data sourcing, transformation, and visualization. This project showcases the implementation of an automated data pipeline using a cryptocurrency API, Python for data processing, and a structured Power BI data model to deliver business‑focused analytics and risk metrics.
